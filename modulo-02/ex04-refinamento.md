# Exercício 4

## Nível 1 — Visão geral

1. Registrar os itens da compra
2. Calcular o subtotal
3. Aplicar o desconto de 5%, caso o cliente tenha cartão fidelidade
4. Mostrar o total a pagar

## Nível 2 — Passos detalhados

1. Iniciar o sistema de caixa
2. Definir o subtotal como zero
3. Ler o preço de cada item da compra
4. Somar o preço de cada item ao subtotal
5. Repetir o processo até que todos os itens sejam registrados
6. Perguntar se o cliente possui cartão fidelidade
7. Se o cliente possuir cartão fidelidade:
    - Calcular o desconto de 5%.
    - desconto = subtotal \* 0,05
    - total = subtotal - desconto
8. Senão:
    - total = subtotal
9. Mostrar o subtotal
10. Mostrar o desconto aplicado, caso exista
11. Mostrar o total a pagar
12. Finalizar o sistema
