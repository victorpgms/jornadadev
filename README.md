## 🎓 Sobre o programa

O **Jornada DEV START** é um programa de formação promovido pela **TOTVS Paulista**, com foco no desenvolvimento de habilidades essenciais para quem deseja atuar com tecnologia e programação.

A jornada aborda conteúdos como lógica de programação, algoritmos, pseudocódigo, fluxogramas, estruturas de decisão, repetição, funções, arrays, banco de dados e desenvolvimento com tecnologias do ecossistema TOTVS.

Este repositório foi criado para organizar meus exercícios, práticas e projetos desenvolvidos ao longo do programa, além de registrar minha evolução durante essa experiência.

## 🎯 Objetivos na Jornada DEV

Durante o programa, pretendo:

- Fortalecer minha lógica de programação;
- Aprender mais sobre o ecossistema de desenvolvimento da TOTVS;
- Conhecer melhor sistemas corporativos e o Protheus;
- Desenvolver soluções utilizando Harbour e ADVPL;
- Praticar organização de código e versionamento com Git;
- Evoluir tecnicamente por meio dos exercícios e projetos;
- Compreender melhor o fluxo de desenvolvimento dentro de uma grande empresa de tecnologia.
