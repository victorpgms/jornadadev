# Exercício 1 — Função ou procedimento? (conceitual)

Para cada item abaixo, responda se é uma função (retorna um valor) ou um procedimento (só
executa uma ação) e justifique em uma frase:

## Str() — converte número em string
R: É uma função, pois o retorno é um dados do tipo string, vindo de um tipo númerico

## QOut() — exibe texto na tela
R: É um procedimento, pois somente exibe texto na tela. Sem retorno.

## Date() — retorna a data atual
R: É uma função, pois retorna a data atual que pode ser reutilizada.

## Len() — retorna o tamanho de uma string ou array

R: É uma Função, pois retorna um valor númerico com o tamanho de um objeto:

Ex: 
   LOCAL aCaixa := {"Pedra", "Papel", "Tesoura"}
   len(aCaixa) ==> O resultado seria o valor reutilizavel 3.

## Upper() — retorna a string em maiúsculas
R: Retorna uma nova string (em maiúsculas) como resultado.

## ClearScreen() — limpa a tela
R: É um procedimento, pois somente limpa os valores da tela, e não tem nenhum retorno.
